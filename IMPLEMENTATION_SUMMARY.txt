使用方法 (How to Use):
======================================

方法 1 - GitHub Actions (推荐):
1. 访问 GitHub Actions 页面
2. 选择 "Build Example Supernote Plugin"
3. 点击 "Run workflow"
4. 等待构建完成
5. 下载 supernote-example-plugin artifact
6. 解压得到 .snpk 文件
7. 传输到设备并安装

方法 2 - 本地构建:
1. cd example-plugin
2. npm install
3. chmod +x buildPlugin.sh
4. ./buildPlugin.sh
5. 查看 plugin-output/*.snpk
